##### 让球和被让球

- 让球一方，为上盘 (走地数据中盘口前有*)
- 被让球一方，称为下盘 

##### 亚盘水位

- 低水区间：0.85以下
	- 超低水：0.75以下
	- 低水：0.75-0.85

- 中水区间：0.85-0.95
	- 中低水：0.85-0.90
	- 中水：0.90-0.95
- 高水区间：0.95以上
	
	- 中高水：0.95-1.00
	- 高水：1.00-1.08
	- 超高水：1.08以上
